Objectif:
Personnaliser l'aspect d'une checkbox.
Utiliser les transitions CSS.

Travail à faire :
Appliquer les styles nécessaires pour personnaliser l'aspect d'un checkbox comme dans le fichier resultat.gif

Indication :
Utiliser La balise <label> qui  vous permet de traiter  le clic sur l'étiquette, comme un clic sur la "checkbox". Il existe deux façons de créer cette association :
* Enveloppez le checkbox par le label
* Associer le label avec le checkbox grâce à l'attribut "for".
